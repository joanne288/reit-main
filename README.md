# REIT Main

A repository for tracking and analyzing Real Estate Investment Trust (REIT) performance and real estate data.

## Overview

This project collects, stores, and analyzes REIT data to provide insights into real estate investment performance. It includes data files, analysis scripts, and reports to help evaluate REIT metrics over time.

## Contents

```
reit-main/
├── reit-main/          # Core data and analysis files
├── README.md           # Project documentation
└── .gitattributes      # Git LFS tracking configuration
```

## Tools & Technologies

- **Microsoft Excel** — Data storage and analysis
- **Git & GitHub** — Version control and file hosting
- **Git LFS** — Large file storage for oversized data files

## Getting Started

1. Clone the repository:
   ```cmd
   git clone https://github.com/your-username/reit-main.git
   ```

2. Make sure Git LFS is installed:
   ```cmd
   git lfs install
   git lfs pull
   ```

3. Open the data files in Excel or your preferred tool.

## Data

The main data file (`reit-main`) contains REIT performance records including:
- Property valuations
- Dividend yields
- Return on investment metrics
- Historical performance data

> **Note:** Large files are managed via Git LFS. Make sure Git LFS is installed before cloning.
